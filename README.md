# 📚 Mecânica Estatística – Listas de Exercícios Resolvidas

Repositório dedicado às resoluções das listas de exercícios da disciplina de **Mecânica Estatística**, desenvolvidas durante o curso de **Mestrado em Física (2026.1)**.

O objetivo deste projeto é reunir soluções detalhadas, apresentando todas as etapas dos cálculos e as interpretações físicas envolvidas, servindo como material de estudo e consulta para estudantes interessados na área.
